JavaScript Basics

Make web pages interactive, dynamic, and smart

🧠 What is JavaScript?

JavaScript is the programming language of the web. It brings websites to life by adding interactivity — handling clicks, updating content without reloading, validating forms, creating animations, and fetching data from servers. Every modern website uses JavaScript!

// Your first JavaScript: show a greeting let name = "Learner"; console.log("Hello, " + name); alert("Welcome to JavaScript!");
💡 Pro tip: Open your browser's Developer Console (F12) and try typing console.log("Hello") — it's the perfect place to test JavaScript snippets live!

📦 Variables & Data Types

Variables store information. Use let for values that change, const for fixed values. JavaScript has numbers, strings (text), booleans (true/false), arrays (lists), and objects. The language is dynamically typed — no need to declare types!

let score = 100; // number const name = "JavaScript"; // string let isFun = true; // boolean let colors = ["red", "blue", "green"]; // array
✅ Best practice: Always use const by default. Only use let when you know the value will change. Avoid var — it's outdated!
🔄 Functions & Events

Functions are reusable blocks of code. They can take inputs (parameters) and return outputs. Combine them with events like click, mouseover, or keypress to make pages respond to user actions.

// Define a function function greet(user) { return `Hello, ${user}!`; } // Run when button is clicked button.addEventListener('click', () => { alert(greet("Student")); });
🚀 Arrow functions: () => {} is a shorter, modern way to write functions. Perfect for callbacks and event handlers!
🌳 DOM Manipulation

The DOM (Document Object Model) represents your HTML as a tree of objects. JavaScript can change elements, styles, attributes, and even create new elements — all without refreshing the page!

// Change text inside an element document.getElementById("title").textContent = "New Heading"; // Add a CSS class element.classList.add("active"); // Create and insert new element let newDiv = document.createElement("div"); newDiv.innerHTML = "I'm dynamic!"; document.body.appendChild(newDiv);
⚡ Modern tip: querySelector() and querySelectorAll() are more powerful than getElementById — they use CSS selectors!
