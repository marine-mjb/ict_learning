HTML & CSS Fundamentals

Build beautiful, responsive websites from scratch

📖 What is HTML?

HTML — HyperText Markup Language — is the skeleton of every webpage. It uses tags to define elements like headings, paragraphs, links, and images. Browsers read these tags and render them visually for the user.

<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is a paragraph.</p> </body> </html>
💡 Pro tip: Every HTML document must start with <!DOCTYPE html>. This tells the browser which version of HTML you're using (HTML5 in modern web development).
<html> <head> <body> <h1>–<h6> <p>
🎨 What is CSS?

CSS — Cascading Style Sheets — is what makes websites look great. While HTML handles structure, CSS handles visual presentation: colors, fonts, spacing, layout, and animations. You can apply CSS in three ways: inline, internal, or external.

/* External CSS (best practice) */ h1 { color: #00d4ff; font-size: 2rem; font-family: 'Arial', sans-serif; } p { color: #334155; line-height: 1.6; margin-bottom: 1rem; }
✅ Best practice: Always keep your CSS in a separate .css file and link it with <link rel="stylesheet" href="styles.css"> in your HTML head. This keeps code clean and maintainable.
📦 The Box Model

Every HTML element is a rectangular box. The CSS Box Model describes the space around it: content (what's inside), padding (space inside the border), border (the edge), and margin (space outside the border).

.card { width: 300px; padding: 20px; /* inside space */ border: 2px solid #00d4ff; margin: 16px; /* outside space */ border-radius: 12px; /* rounded corners */ }
⚠️ Common mistake: By default, width doesn't include padding or border. Add box-sizing: border-box to your CSS so widths are predictable and include all spacing.
📐 Flexbox & Grid

Modern CSS layout uses Flexbox for one-dimensional layouts (rows or columns) and CSS Grid for two-dimensional layouts. These replaced old float-based layouts and make responsive design much easier.

/* Flexbox: center items horizontally & vertically */ .container { display: flex; justify-content: center; align-items: center; gap: 1rem; } /* Grid: 3-column equal layout */ .grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; }
💡 Remember: Flexbox is great for navigation bars, card rows, and centering. Grid shines for full page layouts and gallery-style arrangements. Use both together!
